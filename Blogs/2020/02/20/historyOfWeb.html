The standardization of Java script

    1. Accessibility, no permission from an authority is needed to access or publish onto the web
    2. Non-discrimination based off of wealth
    3. Bottom up design. The code used should be accessible and user friendly to encouraging maximum participation and experimentation.
    4. In order for the web to work, the computers need to speak the same language to communicate
    5. There had to be a consensus oh practices, which came from the W3C.
